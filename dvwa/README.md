# DVWA Web Application Security Lab

## Overview

This lab focuses on practicing common web application attacks in a controlled DVWA environment to understand attack behavior, security risks, and basic mitigation methods.

## Lab Environment

- DVWA
- Kali Linux / Attacker machine
- Web server
- Browser and basic web security testing tools

## Attacks Practiced

- SQL Injection
- Brute Force
- Cross-Site Scripting
- Command Injection
- File Upload vulnerabilities

## What I Learned

- How common web vulnerabilities work in a lab environment
- How to identify basic attack behavior
- How to document attack steps, impact, and mitigation
- How web attacks may appear in logs or security monitoring tools
