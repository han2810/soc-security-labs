# SOC Security Labs

This repository contains my personal cybersecurity labs focused on SOC Tier 1 skills, including web attack analysis, IDS/IPS monitoring, SIEM alert investigation, and basic incident response.

## Labs

### 1. DVWA Web Application Security Lab
Practiced common web application attacks in a controlled DVWA environment, including SQL Injection, Brute Force, XSS, Command Injection, and File Upload vulnerabilities.

### 2. Suricata IDS/IPS and Wazuh SIEM Integration Lab
Configured Suricata IDS/IPS and integrated its alerts with Wazuh SIEM to detect and analyze brute force and SQL Injection attacks against a backend server.

## Skills Practiced

- Web application security testing
- IDS/IPS configuration
- SIEM alert monitoring
- Log analysis
- Alert triage
- Basic incident response
- Security documentation
